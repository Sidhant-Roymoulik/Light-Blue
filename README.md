# Light Blue
A Chess Engine Built in Golang

---

### Basic Requirements

 - [x] DFS
 - [x] Piece Square Tables
 - [x] Move Ordering
 - [x] Alpha-Beta Pruning
 - [x] Quiescence Search
 - [x] Iterative Deepening
 - [ ] Transposition/Hash Tables
 
### Additional Optimizations
